# Smooth8x

Smooth8x is a texture pack created for the purpose of improving performance for low-end computers or for someone who wants to experience Minecraft in a new way.

(Also please note that this texture pack is under All Rights Reserved, and that this repository is for bug report or for suggesting improvements only. If you want to use my texture pack please download it from modrinth as it supports my work. Download Link: https://modrinth.com/resourcepack/smooth8x)

### How does it improve performance?

Smooth8x works by having textures scaled to an 8x8 format instead of the default 16x16. This reduces the consumption of VRAM (Video Random Access Memory), which can lead to better performance or even higher FPS (Frames Per Second) if your GPU (Graphics Processing Unit) is old or does not have a lot of VRAM.

### Installation guide:

1. First, download the texture pack for your Minecraft version.
2. Next, move the .zip file out of your downloads folder, and place it into the "recourcepacks" folder.
3. After that, launch your game and navigate to: "Options -> Resourcepacks" and enable the texture pack.
4. Enjoy the texture pack!

(If you get a warning while enabling the texture pack, that says it is broken or outdated, then you can safely ingnore the warning.)
